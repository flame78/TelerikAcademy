Global.asax: Entry point for application. Define global application events.
Bundele.config : groping and compress CSS styles.
Site.Master: Template for Desktop app.
Site.Mobile.Master: Template for Mobile app.
Web.Config: Contains application configuration.
packages.config: track installed packages and respected versions.
ViewSwitcher.ascx: This control can be used on pages to allow users to switch between desktop and mobile versions of a page. The control can be placed on a page or in the master page for all pages, and the user will have links for switching to mobile version or desktop if viewing on a mobile device.
*.apsx: Web Forms pages with reference to Code-Behind
Code-Behind gives you the ability to separate the code that defines the appearance of a Web page from the code that executes in response to events.
